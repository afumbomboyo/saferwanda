# **App Name**: SafeRwanda

## Core Features:

- SafeRwanda AI Concierge: An AI-powered consultant tool that answers safety-related questions and uses reasoning to advise users on which specific services are right for them.
- Responsive Landing Page: A high-impact landing page featuring the core safety catchphrases and a clear mission statement, optimized for both mobile and desktop users.
- Services Information Hub: A dedicated page detailing the seven core offerings: Child Protection, Elderly Care, Fire Prevention, Property Security, Asset Protection, Neighborhood Surveillance, and Smart Community integration.
- Unified Authentication Flow: A streamlined login and 'Get Started' registration interface to onboard users into the SafeRwanda ecosystem.

## Style Guidelines:

- Primary color: Royal Cobalt (#2E5BFF), a deep, high-tech blue representing reliability and advanced security.
- Background: Obsidian (#0F111A), a dark, elegant base to ensure high contrast for service information and CTAs.
- Accent: Electric Sky Blue (#00B2FF), used for highlighting the 'Get Started' button and smart features.
- Headlines: 'Poppins' for a clean, modern, and professional aesthetic across the landing page.
- Body Text: 'Inter' for maximum legibility when explaining complex safety services.
- Thin-stroke, security-grade SVG icons representing each of the 7 services for high visual clarity.
- A mobile-first, responsive bento-grid layout for the services page to ensure a premium desktop and mobile experience.
- Subtle fade-in effects for service cards and smooth scrolling transitions between landing page sections.
